== Description ==


Based on WP-CopyProtect(Basic) I'v made plugin to prevent text selection 
or right-button click. 
The basic's of it that we are redefining functions with JavaScript, on 
actions that we need to block, 
they all must return false; There are few common plugins on wordpress 
repository but almost all of them, 
have right-click alert notice, and it's annoing... So I present 
Posts-Protect-Quiet...

На базе WP-CopyProtect(Basic) сделал плагин который не выводит никаких 
предупреждений при нажатии 
правой кнопкой мыши или выделении текста. Основаная суть плагина, это 
переопределение функций которые нам 
нужны так чтобы они всегда возвращали false; И делов то. Таких плагинов 
на сайте Wordpress довольно много и 
сделаны они по одной схеме. Только данное отличие скрипта, что он не 
информирует о производимых действиях 
назойливыми alert сообщениями...
